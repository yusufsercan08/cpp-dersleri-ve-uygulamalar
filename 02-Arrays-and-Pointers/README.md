# 📁 02-Arrays-and-Pointers

This folder contains practical exercises and projects focusing on C++ arrays, memory manipulation, shifting algorithms, and pointer mechanics.

## 📜 Files & Descriptions

* **`array_basics.cpp`**: Introduction to 1D/2D array declarations, iteration, and element accessing.
* **`pointer_fundamentals.cpp`**: Demonstrates pointer syntax, memory address referencing (`&`), and dereferencing (`*`).
* **`array_shifting_algorithms.cpp`**: Array element rotation, left/right shifting, and memory re-indexing logic.
* **`dynamic_memory_allocation.cpp`**: Usage of `new` and `delete` keywords for dynamic array allocation on the Heap.
