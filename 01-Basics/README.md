# 📂 01-Basics

This folder contains foundational C++ practice codes including basic I/O operations, conditional logic, functions, and algorithms.

## 📜 Files & Descriptions

* **`circle_area_perimeter.cpp`**: Calculates the area and perimeter of a circle based on user-input radius.
* **`functions_1.cpp`**: Demonstrates basic C++ function definition, declaration, and scope.
* **`functions_2.cpp`**: Advanced function examples including parameter passing.
* **`functions_3.cpp`**: Password verification system utilizing `std::string::compare()`.
* **`functions_4.cpp`**:An educational example featuring Saybot battery voltage monitoring and a function with default parameters.
* **`menu_based_calculator.cpp`**: Interactive command-line calculator using control flow and arithmetic operations.
* **`perfect_square_check.cpp`**: Checks whether a given integer is a perfect square.
